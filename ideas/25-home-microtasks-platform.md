# پلتفرم سفارش کارهایی که در خانه انجام می‌شود (Micro-task Home Economy)

## Problem
بسیاری از کارهای کوچک و روزمره در ایران وجود دارند که:
- ارزش اقتصادی دارند اما بازار رسمی ندارند
- به‌صورت پراکنده و غیرساختاریافته انجام می‌شوند
- اعتماد و قیمت‌گذاری مشخص ندارند

مثال‌ها:
- بسته‌بندی ساده
- آماده‌سازی محصولات خانگی
- کارهای سبک اداری یا دیجیتال

نتیجه:
- اتلاف ظرفیت نیروی کار خانگی
- نبود درآمد مکمل برای خانواده‌ها
- عدم اتصال عرضه و تقاضا

## Target Users
- خانواده‌ها و افراد خانه‌دار
- کسب‌وکارهای کوچک
- فروشگاه‌های آنلاین
- تولیدکنندگان خرد

## Core Idea
یک پلتفرم برای تبدیل کارهای کوچک خانگی به micro-taskهای قابل سفارش:
- تعریف taskهای ساده و استاندارد
- اتصال سفارش‌دهنده و انجام‌دهنده
- پرداخت بر اساس task

## Solution Overview
- لیست taskهای استاندارد قابل انجام در خانه
- سفارش‌دهی ساده
- تایید انجام کار
- پرداخت پس از تایید

## Key Features
- marketplace برای micro-taskها
- تعریف استاندارد برای هر task
- پروفایل انجام‌دهندگان
- سیستم تایید کیفیت
- پرداخت مرحله‌ای

## Value Proposition
- ایجاد درآمد انعطاف‌پذیر برای افراد
- کاهش هزینه عملیات برای کسب‌وکارها
- استفاده بهینه از ظرفیت خانه‌ها

## Differentiation (Iran Context)
- مناسب اقتصاد غیررسمی و خانگی
- قابل اجرا بدون زیرساخت پیچیده
- ترکیب trust + simple workflow

## MVP Scope
- ثبت task
- پذیرش توسط کاربران
- تایید انجام
- پرداخت ساده

## Future Extensions
- اتصال به O-014 (dispute resolution)
- rating مبتنی بر رفتار
- دسته‌بندی هوشمند taskها

## Risk / Constraints
- کنترل کیفیت خروجی
- اختلاف در تعریف انجام صحیح کار
- scalability مدیریت taskها

Gig Economy / Micro-task Platform / Home Economy
