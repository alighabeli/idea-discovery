# مارکت‌پلیس عمده واقعی (نه دایرکتوری)

## Problem
بازار عمده‌فروشی ایران عمدتاً غیرشفاف و رابطه‌محور است. کشف تامین‌کننده مناسب سخت است و اغلب:
- قیمت‌ها شفاف نیست
- موجودی واقعی مشخص نیست
- معامله خارج از پلتفرم انجام می‌شود

نتیجه:
- ناکارآمدی در زنجیره تامین
- وابستگی به واسطه‌ها
- عدم کشف بهینه عرضه‌کننده

## Target Users
- عمده‌فروشان
- خرده‌فروشان
- واردکنندگان کوچک
- پخش‌کنندگان

## Core Idea
یک مارکت‌پلیس واقعی B2B عمده که فقط لیست نیست، بلکه:
- موجودی واقعی را نمایش می‌دهد
- قیمت‌ها را قابل مقایسه می‌کند
- امکان سفارش مستقیم دارد

## Solution Overview
- لیست محصولات عمده با موجودی لحظه‌ای
- پروفایل تامین‌کننده
- سیستم سفارش مستقیم
- حداقل استاندارد برای ثبت کالا

## Key Features
- کاتالوگ محصولات عمده
- نمایش قیمت و موجودی
- سفارش‌گذاری مستقیم
- امتیازدهی تامین‌کننده
- فیلتر منطقه/قیمت

## Value Proposition
- حذف واسطه‌های غیرضروری
- افزایش شفافیت قیمت
- کاهش هزینه تامین کالا

## Differentiation (Iran Context)
- تمرکز روی بازار واقعی عمده‌فروشی ایران
- طراحی برای فروشندگان سنتی
- ترکیب آنلاین + عملیات آفلاین

## MVP Scope
- ثبت محصول
- نمایش لیست عمده
- ثبت سفارش
- پروفایل فروشنده

## Future Extensions
- اتصال به سیستم اعتبار
- پیش‌بینی قیمت
- مدیریت لجستیک

## Risk / Constraints
- مقاومت عمده‌فروشان سنتی
- مشکل تامین داده واقعی

## Category
B2B Marketplace / Supply Chain / Wholesale Commerce
